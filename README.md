# Principais Comandos para GitHub e GitFlow

## Clonar repositório do GitHub
git clone <https://github.com/Tryss-mp3/Git-Flow---Testes.git>

## Inicializar repositório
git init

## Informações e mudanças dos arquivos
git status

## Adicionar arquivos
git add

## Realizar o commit das mudanças feitas em arquivos
git commit

## Buscar alterações
git pull origin main

## Enviar minhas alterações para o repositório
git push origin main

## Ramificações (branchs) do repositório
git branch

## Navegar entre diferentes versões
git switch ou git checkout

## Mesclar mudanças de branches em uma única branch
git merge

## Desfazer alterações dos commits
git reset
